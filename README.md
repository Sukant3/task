Angular Posts App (Task)

A simple Angular application that demonstrates fetching and posting data using HttpClient with a clean UI and no external CSS libraries.

This project uses the public mock API JSONPlaceholder to fetch and add posts.
__________________________________________________________________________________________________________


Tech Stack

Angular (Standalone Components)

TypeScript

HTML

CSS (No Bootstrap / No UI libraries)

JSONPlaceholder API
